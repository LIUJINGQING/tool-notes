## 1. 基本概念

- 仓库 (Repository)

  仓库用来存放项目代码，每个项目对应一个仓库，多个项目则对应多个仓库



- 收藏 (Star)

  收藏项目，方便下次查看。

  

- 复制克隆项目 (Fork)

  克隆原有仓库为自己独立的仓库



- 发起请求 (Pull Request)

  像
  
  - ==注意==`dfklasj`



- [ ]  的说法
- [x]  交付水电费



- [ ] 发斯蒂芬

- [ ] dfasd
- [x] 发顺丰


- 关注 (Watch)

  关注项目，当项目更新可以接收到通知提醒

  

- 事务卡片 (Lssue)
